Logger 工具类分享

前言

稍大一些的系统设计，日志模块我们一般都会选用 log4net 或 NLog，开源并且功能齐全，配置功能很好用。

但是偶尔我们也会开发一些小工具，只是简单的输出一些日志，如果再选择这些开源项目的方案，不免显得有点“沉重”。

后面就针对这简单的使用场景，写了一个“简陋”的工具类 Logger.cs 满足一些简单场景的使用。
